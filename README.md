# HeatmapCalculator
Модуль предназначен для расчёта тепловых карт на основе различных параметров.
Картографические данные получаются при помощи Overpass API.

### Features
- Поддержка вывода результатов в различных форматах(CSV, GeoJSON, XML)(GeoJSON и XML в разработке)
- Выбор фильтров для отрисовки (По кол-ву различных обьектов вблизи домов: магазинов, учебных заведений, медицинских заведений)(В разработке)
- Выбор функции расчёта (Метод Хаффа и другие)(В разработке)
